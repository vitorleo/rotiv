# v0.2.3 Changelog

Released: 2026-03-15

## Summary

Patch release addressing the persistent `Cannot find package 'tsx'` crash that blocks `rotiv dev`
and `rotiv migrate` when `@rotiv/*` npm packages are not installed (and therefore the project's
`node_modules` is empty).

---

## Bug Fixes

### `rotiv dev` / `rotiv migrate` crash: "Cannot find package 'tsx'"

**Root cause:** Both commands invoke `node --import tsx`, which resolves `tsx` from the project's
`node_modules`. When `pnpm install` cannot run (because `@rotiv/*` packages are not on npm), the
project's `node_modules` is empty, so Node.js cannot find `tsx`.

**Fix:** Added `resolve_tsx_loader(project_dir)` to both `worker.rs` and `migration.rs`.
This function searches for tsx at:
1. `<project_dir>/node_modules/tsx/dist/esm/index.cjs` (user's installed tsx)
2. `<binary_dir>/node_modules/tsx/dist/esm/index.cjs` (bundled with rotiv)
3. `<binary_dir>/../node_modules/tsx/dist/esm/index.cjs`
4. Falls back to bare `"tsx"` (globally installed)

When tsx is found, its absolute path is passed to `node --import` instead of the bare name,
bypassing Node.js's module resolution entirely.

**Improved error message:** When drizzle-kit output is empty, the error now says
"tsx or drizzle-kit may not be installed (run: npm install -g tsx)" instead of just "no output".

---

## Installer

`install.sh` now:
- Verifies that `node` is in PATH before proceeding (exits with error if not)
- Checks for `tsx`; if not found, automatically runs `npm install -g tsx`
- Displays node and tsx versions in the success message

---

## Note on npm packages

`@rotiv/*` packages are still not on the npm registry (pending `NPM_TOKEN` secret configuration
in GitHub Actions). The fixes above reduce the impact of this: `rotiv dev` and `rotiv migrate`
will now work as long as `tsx` is available globally or in any of the searched locations.

The full fix chain for a working app remains:
1. Publish `@rotiv/*` to npm (set `NPM_TOKEN` secret)
2. Run `pnpm install` in the project
3. `rotiv migrate` + `rotiv dev`
